Energy Efficiency

Great that you are here and want to learn more about green code's key performance indicators (KPIs)!
This KPI deals with is energy efficiency. Energy efficiency is often measured by the energy a code, program, or software consumes at runtime on the CPU level (in Joules).
You want to learn more about different ways of improving the energy efficiency of your code in just a few steps? Let's get started!

Data Structures

What is a data structure?

A data structure is a format for organizing, processing, retrieving, and storing data

It is an important concept as it allows to access and work with data appropriately

Each data structure contains information about the data values, relationships between the data, and (in some cases) functions that can be applied to the data (e.g., sort the data or add specific values)

Most modern programming languages have built-in data structures to choose from (e.g., Arrays, Lists, Maps, Sets, etc.)

A reasonable choice of the data structure is very dependent on the problem you are dealing with (e.g., what data will be stored, which operations should the data structure provide, etc.)

What do data structures have to do with green code?

A poor choice of the data structure in terms that it is not suitable for the data, the operations you want to perform on the data, and the program can cause severe drawbacks in the energy consumption, and thus the energy efficiency of your program

A study from Pereira and colleagues in 2016 showed that an appropriate choice of the data structure can improve the energy consumption by up to 11%

How can I measure the quality of a data structure with regards to energy efficiency?

As there are many different programming languages and data structures, an assessment must always be made on a case-by-case basis

Nevertheless, there are two ways of easily assessing the energy consumption of your data structures when using the Java Collection Framework

  • Click here to see energy consumption tables of Maps, Sets, and Lists (when storing String data with a size of 25k) as well as their respective operations to help you choose a suitable way of storing your data
  • Click here to learn more about Java Collection Performance, a framework that provides code (i.e., an interface) for testing the performance of other data structures on your own
